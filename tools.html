Tools

In this assessment, we have used a number of tools to develop, collaborate, and communicate on the project.

  • Discord: An instant messaging platform with private chats and community servers.
    • We used Discord to assembled our group, find each other on a community server setup by another student for the unit.
  • Microsoft Teams: A collaboration environment allowing users to share thoughts and files.
    • We used MS Teams to have weekly meetings and discuss the tasks at hand
    • Shared files and feedback using the programs file sharing functionality
  • GitHub: A code hosting platform for version control and collaboration.
    • We used GitHub to keep the website code under version control, and to allow all members to have access.
    • We also used the 'Pages' functionality of GitHub to host our project as a website.
  • Microsoft Word: A word processing software.
    • We used Microsoft Word to develop our reports and research.
  • Visual Studio Code: A source-code editor for use with a variety of programming languages.
    • We used Visual Studio Code to develop our websites HTML and CSS

Primarily, we have used a GitHub repo (https://github.com/s3930650/Assignment2) to edit our website as sections of the project have been completed. Primarily Leigh has looked after this, but everyone in the group has had access and have edited sections when needed. Due to this, commits are a mix of small edits correcting spelling mistakes or style changes, and also large commits with HTML documents fleshed out with the content of our work. When it came to editing and sharing our work, we utilised Microsoft Teams file sharing as it provided another central place to read and write documents, and unlike GitHub you could see live changes within the editor. This proved useful in multiple cases, such as in meetings when we could discuss edits whilst making them, and also during our interview with a professional as we could strike out questions he had answered to avoid asking them twice.
